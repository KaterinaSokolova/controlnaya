В данной программе представлено решение следующей задачи:
Написать программу, которая из имеющегося массива строк формирует новый массив из строк, 
длина которых меньше, либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. 
При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

Этапы решения:
1. Запрос от пользователя длины начального массива
2. Запрос от пользователя элиметов начального массива
3. Вывод начального массива на экран
4. Расчет длины искомого массива, путем проверки элиментов начального массива на соответствие необходимой длине
5. Вывод элиментов искомого массива на экран,путем проверки элиментов начального массива на 
соответствие необходимой длине
